## HOW THE AUTOMATION WORKS

Every match fires the **moment-engine.js** pipeline:

```
Match event detected (API-FOOTBALL polling every 90s)
  → moment-engine.js identifies event type (GOAL / RED_CARD / ELIMINATION / etc.)
  → Fires up to 18 simultaneous actions within 60 seconds
  → world-engine.js generates AI content in 28 languages via Groq
  → OneSignal web push to all subscribers
  → Beehiiv newsletter auto-draft queued
  → Pinterest/Reddit/Telegram bots post
  → Affiliate links served geo-correctly by affiliate-matrix.js
  → self-improvement-loop.js logs results, A/B tests CTAs
```

**You don't touch anything.** The systems run 24/7 on Railway.app.
