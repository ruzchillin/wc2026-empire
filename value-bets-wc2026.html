📐 How We Find Value

The Value Formula

A bet has "value" when the bookmaker's implied probability is lower than our estimated true probability. For example:

Value% = (True Probability × Decimal Odds) − 1
e.g. Morocco at 80/1: (0.03 × 82) − 1 = 1.46 = +146% edge

Our AI model uses Elo ratings, recent form, squad depth, draw probability in knock-out stages, manager track record, and set-piece data to estimate true probability. Edge values above +10% are worth backing.
