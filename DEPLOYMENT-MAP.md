## PART 1: THE 5 RAILWAY SERVICES (Your AI Agents)

These are the 5 processes that run 24/7 on Railway. Think of each as a separate "worker" doing a different job.

```
Railway Account
├── Service 1: telegram-bot        (always-on, $5/month)
├── Service 2: world-engine        (always-on, $5/month)
├── Service 3: autonomous-agent    (always-on, $5/month)
├── Service 4: whatsapp-bot        (always-on, $5/month)
└── Service 5: daily-briefing      (cron job, free tier)
```

### Service 1 — `telegram-bot`
**File:** `bot-commands.js`
**What it does:** Responds to every Telegram command (/predict, /fatigue, /referee, /odds, etc.) and handles all 18+ bot commands.
**Also loads:** `recommendations-engine.js` (the income stream advisor)
**Runs:** Always-on (polling mode, checks every second for new messages)
**Cost:** ~$5/month on Railway Starter plan

### Service 2 — `world-engine`
**File:** `world-engine.js`
**What it does:** The intelligence brain — 14 modules scanning news, tracking rumours, monitoring odds movements, generating SEO pages on breaking events, translating content to 12 languages, tracking injuries/transfers/drama.
**Heartbeat:** 1-minute main loop with sub-loops at 3min/6min/12min/60min
**Cost:** ~$5/month

### Service 3 — `autonomous-agent`
**File:** `autonomous-agent.js`
**What it does:** The 24/7 operator — auto-responds to subscriber Telegram messages, detects goals and broadcasts them, posts 3x/day automatically, runs health checks, tracks milestones, detects traffic spikes, runs the A/B learning cycle.
**Heartbeat:** 10-second main loop
**Cost:** ~$5/month

### Service 4 — `whatsapp-bot`
**File:** `whatsapp-bot.js`
**What it does:** Express.js webhook server for WhatsApp Business API — receives incoming WhatsApp messages and serves the interactive menu, AI picks, live scores, goal alerts.
**Runs:** Always-on HTTP server (port 3002), registered as Meta webhook
**Cost:** ~$5/month

### Service 5 — `daily-briefing` (Cron)
**File:** `daily-briefing.js`
**What it does:** Runs once per day at 8am UTC — sends the full morning briefing to your Telegram with yesterday's results, today's picks, subscriber growth, revenue summary.
**Railway cron config:** `0 8 * * *`
**Cost:** Free (Railway includes cron jobs)

## PART 5: ALL 14 AUTOMATED WORKFLOWS (Running in Parallel)

This is what the system is actually doing at any given second:

### Every 10 seconds
- `autonomous-agent.js` → polls Telegram for new messages → auto-responds if needed

### Every 60 seconds
- `autonomous-agent.js` → checks API-Football for new goals/red cards → if found, broadcasts to all platforms

### Every 3 minutes
- `world-engine.js` → scans NewsAPI + BBC RSS + Goal.com RSS for breaking news → classifies viral potential → queues drama/SEO/alerts

### Every 5 minutes
- `autonomous-agent.js` → checks if it's a scheduled post time (8am/2pm/9pm UTC) → generates and posts preview/prediction/recap

### Every 5 minutes
- `autonomous-agent.js` → checks if any milestone was hit (subscriber count, revenue threshold) → auto-posts celebration

### Every 6 minutes
- `world-engine.js` → checks The Odds API for line movements ≥0.15 on WC matches → if found, premium alert

### Every 12 minutes
- `world-engine.js` → asks Groq for top 5 trending WC searches today → queues proactive SEO pages

### Every 15 minutes
- `autonomous-agent.js` → pings Groq API, API-Football, Telegram → sends alert if any service is down

### Every 60 minutes
- `autonomous-agent.js` → checks Vercel Analytics for traffic spikes (2×+ normal) → if spike detected, generates content around trending topic
- `world-engine.js` → generates press-ready story from prediction stats → queues for media outreach

### Every hour (WhatsApp)
- `whatsapp-bot.js` → checks if any drip messages are due for the day → sends to subscribers who hit milestone

### Every 3 hours
- `self-improvement-loop.js` → analyzes A/B test results → picks winners → applies to future content

### 8am UTC daily
- `daily-briefing.js` (Railway cron) → compiles overnight results + today's fixtures → sends full briefing to your Telegram

### 9am UTC daily
- `recommendations-engine.js` → assesses all income stream statuses → asks Groq for top 3 ROI actions → sends to Telegram + queues smart question

### 11pm UTC daily
- `self-improvement-loop.js` → assembles full nightly optimization report → applies all A/B winners → sends summary to Telegram
